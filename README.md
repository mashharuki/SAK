# SAK

SAK(SuiAgentKit) repo

## How to work

### install

```bash
cd sui-agent
pnpm install
```

### start

```bash
pnpm run dev
```

Swagger UI - [http://localhost:4111/swagger-ui](http://localhost:4111/swagger-ui)

Web App - [http://localhost:4111](http://localhost:4111)

## Tech Stack

- **Mastra**
- **Gemini**
